Biodiesel Calculator (Palmolein & Popular Oils)

A simple CustomTkinter GUI application that helps calculate the required amounts of alcohol and catalyst for biodiesel production from different oils.  
This project is designed for educational purposes and demonstrates how chemistry concepts (molar ratios, densities, catalyst percentages) can be applied in a Python GUI.

- Choose oil type from a list of popular oils (Palmolein, Soybean, Sunflower, Coconut, Olive).
- Select catalyst (NaOH or KOH).
- Select alcohol (Methanol or Ethanol).
- Enter oil amount (ml) and instantly calculate:
  - Required alcohol (ml)
  - Required catalyst (g)
- Dark theme with large, readable text (font size 36 everywhere).
- Compact and user-friendly interface.
